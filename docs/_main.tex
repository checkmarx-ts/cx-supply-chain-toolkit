\chapter*{Foreword}

Supply chain vulnerability scanning has become an important part of managing risk when software
utilizes open source shared components.  Generating the software's tree of dependencies is typically facilitated
by executing the development tools the development tools typically used to produce the compiled software
deployment packages.  In some integration scenarios, it can be challenging to obtain an accurately
generated dependency tree.

This toolkit is designed to make it easier to generate dependency trees without the need to make significant
changes to an existing build environment.
